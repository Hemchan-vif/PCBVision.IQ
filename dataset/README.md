# PCBVision.IQ Dataset

This project utilizes a PCB defect dataset derived from the publicly available DeepPCB benchmark dataset.

## Dataset Classes

- Copper
- Mousebite
- Open Circuit
- Pin-hole
- Short Circuit
- Spur

## Dataset Structure

train/
valid/
test/

## Download

The dataset can be accessed through the PCBVision.IQ Roboflow workspace.

Roboflow Project:
PCBVisionIQ Industrial Defect Dataset

Download the dataset and place it inside:

dataset/
├── train/
├── valid/
├── test/
└── data.yaml
